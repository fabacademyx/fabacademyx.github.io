FAB Academy X

I'm hosted with GitHub Pages. Sesion 1: Principios y prácticas: Presentacion en castellano Gestión de proyectos: Github Materiales y metodologías: gestión de proyectos, documentación y sharing. Assignment: Crear un repositorio en Github con: introduccion/bio y propuesta de proyecto final Sesion 2: Diseño asistido por ordenador: más conocido como CAD Design, es el aprendizaje de software de dibujo 2D y de modelado 3D para realizar un objeto en 3 dimensiones. Assignment: Crear un diseño en 2d o 3d para ser fabricado. Documentarlo online Sesion 3: Corte controlado por ordenador: se aprenderá el uso la cortadora láser aplicada a un proyecto específico. Sesion 4: Escaneo e impresión en 3D: se aprenderá el uso de un scanner y de una impresora 3D ademas del tratamiento digital de las imágenes, el software y el tipo de archivos utilizados por cada máquina. Sesion 5: Programación con Arduino (INCLUYE: dispositivos de entrada y dispositivos de salida): se aprenderá el uso de un sistema embebido, en este caso arduino, y la programación del mismo. Sesion 6: Máquinas controlada por ordenador + Moldes y vaciado (introducción): se aprenderá el uso de las demás maquinarias de control numérico en proyectos de gran escala. Además se hará una introducción al módulo de resinas y componentes para fabricar moldes y replicar objetos. Sesion 7: Aplicaciones e implicaciones de la Fabricación digital. Introducción al concepto y el ethos de un FabLab y organización del mismo. Desarrollo de proyectos finales Sesion 8: Finalización de presentación de proyectos
